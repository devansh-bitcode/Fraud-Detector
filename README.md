*Fraudster is an AI-driven fraud detection system designed to analyze financial transaction data and identify suspicious activity with high accuracy. By learning from historical patterns, the system can effectively flag unusual behavior, helping users quickly spot high-risk transactions. Fraudster showcases practical experience in data preprocessing, feature engineering, visualization, and deploying ML/AI models for real-world fraud detection tasks.*
